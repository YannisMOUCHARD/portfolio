Mini projet SQL

Projet


Objectif :
Notre projet visait à concevoir une base de données pour une société fictive en utilisant SQL et Oracle, incluant la création de tables, de relations et de requêtes pour l'exploitation des données.

Méthodologie :
Nous avons analysé les besoins, conçu les MCD et MLD avec Looping, puis implémenté et alimenté la base de données dans Oracle. Enfin, nous avons développé des requêtes SQL pour manipuler les données.

Difficultés :
La principale difficulté a été la gestion des relations entre les tables, nécessitant une compréhension approfondie des cardinalités.

Compétences acquises :
Ce projet m'a permis d'acquérir des compétences en conception de bases de données, en SQL et en utilisation d'Oracle.
