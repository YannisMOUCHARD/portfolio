Gestion de fichier PYTHON

Projet



Dans le cadre de la Saé "Gestion de fichier en Python", j'ai eu l'opportunité de travailler sur un jeu de données intéressant et pertinent pour le domaine de la santé. Le fichier "comorbidites.json", disponible sur Updago, contient des informations sur les comorbidités associées aux pathologies, traitements chroniques ou épisodes de soins repérés par la Caisse nationale de l’Assurance Maladie (CNAM) en France.



Le jeu de données est structuré de manière à fournir des informations détaillées sur les pathologies et les comorbidités associées.

L'objectif de la Saé était de transformer le fichier Json fourni pour obtenir un fichier csv avec les colonnes suivantes et uniquement pour les lignes où ces colonnes sont complètes et sans données aberrantes : PathologieLaPlusPrécise, NomComorbidité, Année, EffectifPathologie, EffectifComorbidité, Proportion, niveauPrioritaire.

Pour ce faire, j'ai réalisé un script Python "transformerJsonToCsv.py" qui permet de réaliser cette transformation. J'ai pris soin de respecter les bonnes pratiques de programmation (commentaires, indentation, découpage modulaire si besoin) et de produire un fichier csv lisible et exploitable dans Excel.



Télécharger transformerJsonToCsv.py



Ce code Python permet de rechercher le fichier "comorbidites.json" dans plusieurs emplacements courants tels que le bureau et le dossier de téléchargement de l'utilisateur. La fonction "find_comorbidites_file()" parcourt chaque emplacement et vérifie si le fichier existe à l'aide de la fonction "os.path.isfile()". Si le fichier est trouvé, la fonction renvoie son chemin d'accès complet. Sinon, elle renvoie "None".

Le code principal appelle la fonction "find_comorbidites_file()" pour rechercher le fichier et stocke le résultat dans la variable "file_path". Si le fichier est trouvé, le code charge les données JSON depuis le fichier à l'aide de la fonction "json.load()".

Ensuite, le code vérifie que la liste de données contient des dictionnaires avec un champ "fields". Si c'est le cas, le code définit les noms de colonnes à extraire dans la variable "selected_columns". Il trie ensuite les données par plusieurs colonnes à l'aide de la fonction "sorted()" et de la méthode "get()" des dictionnaires.

Le code traite ensuite la colonne "niveau_prioritaire" en remplaçant ses valeurs par la valeur la plus élevée pour chaque entrée. Enfin, le code écrit les données triées dans un fichier CSV avec un point-virgule comme délimiteur à l'aide de la classe "csv.DictWriter". Il vérifie également que toutes les colonnes sélectionnées ont des valeurs non vides avant d'écrire une entrée dans le fichier CSV.

Si le fichier "comorbidites.json" n'est pas trouvé, le code affiche un message d'erreur. Sinon, il affiche un message de succès et ouvre le fichier CSV avec l'application par défaut à l'aide de la fonction "subprocess.run()".
