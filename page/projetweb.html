Projet


Objectif :
Notre projet consistait à réaliser un site web dynamique avec des graphiques interactifs alimentés par une base de données, le thème étant libre. Nous avons choisi de créer un site dédié au football, utilisant les données de Transfermarkt pour présenter les profils des joueurs et simuler des transferts.

Méthodologie :
Nous avons opté pour le thème du football et avons structuré le site en plusieurs pages avec différentes approches. Certaines pages étaient dédiées à la présentation de graphiques et de tableaux, tandis que d'autres favorisaient l'interactivité avec les utilisateurs, incluant un forum de discussion, un blog, et une page affichant la météo actuelle grâce à une API. Nous avons également développé un backend en Python pour gérer le chargement des données dans les tables de la base de données. La gestion de la quantité importante de données a été un défi majeur, nécessitant une optimisation pour assurer des performances fluides.

Difficultés :
La plus grande difficulté a été de gérer la quantité massive de données tout en rendant le site adapté à toutes les tailles d'écrans, assurant ainsi une expérience utilisateur optimale sur divers appareils.

Compétences acquises :
Ce projet m'a permis d'améliorer mes compétences en programmation web et de mieux comprendre le fonctionnement d'une base de données en arrière-plan d'un site. J'ai également acquis de l'expérience dans l'intégration et la gestion de grandes quantités de données externes, comme celles de Transfermarkt, pour enrichir le contenu du site. J'ai aussi appris à optimiser les performances pour gérer efficacement de grands volumes de données.
